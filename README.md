# python-cli-bazel-pytest-sieve-atkin

## Description
Given a number find all primes.
POC for pytest with bazel.

## Tech stack
- python
  - pytest
- bazel

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
